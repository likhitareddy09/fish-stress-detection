# Fish Stress Detection

An AI + Hardware based system for monitoring fish stress in aquaculture environments.

## Modules

### AI Module
- Fish Detection using YOLO
- Fish Tracking
- Fish Count Analysis
- Movement Analysis

### Hardware Module
- Water Temperature Sensor
- pH Sensor
- Dissolved Oxygen Sensor
- Real-time Monitoring
